Introduction Programing Reference

category

Window

name

名前

Window.background

Description

説明

                ウィンドウの背景色を変更するために使われます。

                色はカラーコード（例#000000）、色名（例red）またはcolor関数を使って指定できます。画像は指定できません。

Examples

サンプル

                        
                            import IP

                            window = IP.Window()
                            window.background("red")
                            window.show()
                        
                    
                        
                            import IP

                            window = IP.Window()
                            window.background("#b0c4de")
                            window.show()
                        
                    
                        
                            import IP

                            window = IP.Window()
                            window.background(color(0,128,128))
                            window.show()

Syntax

構文

  • Window.background(color)

Parameters

引数

  • color

    (str)

    必須。任意の色(カラーコード、色名)

  • (color)

    color関数で指定する色

Return

戻り値

void

関連
