name

名前

Ellipse

Description

説明

                ウィンドウ上に楕円もしくは正円を描画するために使用されます。

                指定された座標x,yを中心とした幅width高さheightの楕円を描画します。

                図形の色はfill関数で、塗りつぶしなしはnoFill関数で指定することができます。
                枠線の色はoutlineFill関数で、枠線なしはnoOutline関数で指定できます。
                枠線の太さはoutlineWidth関数で指定可能です。

                楕円の位置は座標を指定する以外にrotate関数を使用することで変化させることができます。ウィンドウの左上座標0,0もしくはchangeBasePoint関数で任意に設定された座標を中心とした円周上を移動します。
