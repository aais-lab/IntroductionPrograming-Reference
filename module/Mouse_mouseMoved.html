name

名前

@mouseMoved

Description

説明

                マウスの動きに応じた処理を行う関数に付与するデコレータです。
                このデコレータを付与した関数は、マウスが移動した際に実行されます。
                この時、マウスの何らかのボタンが押下されている必要はありません。
