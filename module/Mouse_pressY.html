name

名前

mouse.pressY

Description

説明

                何らかのマウスボタンを押下したウィンドウ上のマウスカーソルの座標のうち、Y座標がint型で格納されています。

                何らかのボタンが押下されるまでの初期座標は0で、マウスボタンがウィンドウ上で押下されたタイミングでマウスカーソルのY座標に更新されます。
                次に何らかのボタンが押下されるまでは、その前にボタンが押下されたY座標が格納されています。
                座標はウィンドウの左上隅を基準点（X, Y = 0, 0）とする絶対座標系です。

サンプル
