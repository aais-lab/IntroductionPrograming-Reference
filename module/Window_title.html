category

Window

name

名前

Window.title

Description

説明

                ウィンドウのタイトルを変更するために使用されます。

                マークダウン等のプレーンテキスト以外の文字列は無効になり、プレーンテキストとして表示されます。

Examples

サンプル

                        
                            import IP

                            window = IP.Window()
                            window.title("")
                            window.show()
                        
                    
                        
                            import IP

                            window = IP.Window()
                            window.title("サンプル2")
                            window.show()

Syntax

構文

  • Window.title(title)

Parameters

引数

  • title

    (str)

    必須。ウィンドウタイトル

Return

戻り値

void
