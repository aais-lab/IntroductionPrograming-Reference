name

名前

Line

Description

説明

                ウィンドウ上に直線を描画するために使用されます。

                指定した開始点startXstartYから終了点endXendYまでを一直線に繋ぎます。
                
                線の太さは引数lineWeightか、lineWeight関数で指定することができます。
                枠線を指定することはできません。outlineFill関数、noOutline関数、outlineWidth関数を実行した場合、NotFoundFunctionが送出されます。
                色はfill関数で指定することができます。noFill関数を実行した場合、描画された線は見えません。

                直線の位置は座標を指定する以外にrotate関数を使用することで変化させることができます。ウィンドウの左上座標0,0もしくはchangeBasePoint関数で任意に設定された座標を中心とした円周上を移動します。また、移動時に角度も変化します。角度のみを変化させたい場合はchangeBasePoint関数で直線の重心座標を指定してください。

Examples

サンプル

                        
                        
                            import IP

                            window = IP.Window()
                            line = IP.Line(120,80,340,300)
                            window.show()
