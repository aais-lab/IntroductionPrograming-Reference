Introduction Programing Reference

category

Window

name

名前

Window.size

Description

説明

                ウィンドウの大きさを変更するために使用されます。

                幅widthと高さheightを正の整数で指定できます。
                上限値は幅width1000px、高さheight700pxです。上限を超えて指定した場合、上限値が適用されます。

Examples

サンプル

                        
                            import IP

                            window = IP.Window()
                            window.size(500,500)
                            window.show()

Syntax

構文

  • Window.size(width, height)

Parameters

引数

  • width

    (int)

    1〜1000

    必須。ウィンドウの幅

  • height

    (int)

    1〜700

    必須。ウィンドウの高さ

Return

戻り値

void

関連
