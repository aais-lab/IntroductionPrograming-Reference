category

Mouse

name

名前

@mouseDragged

Description

説明

            マウスの何らかのボタンを押下した状態で動かした際に処理を行う関数に付与するデコレータです。
            このデコレータを付与した関数は、マウスの何らかのボタンを押下しながら動かした（マウスの現在座標が変更された）際に実行されます。
            システム上ではisPressed変数がTrueの場合にのみ発火する@mouseMovedです。
現在の座標はX、Yに、直前の座標がbeforeX、beforeYに格納されています。
