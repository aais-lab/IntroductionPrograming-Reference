category

Keyboard

name

名前

@keyPressed

Description

説明

                キーボードのなんらかのキーを押下した際に処理を行う関数に付与するデコレータです。
                このデコレータを付与した関数は、キーボードのなんらかのキーが押下された際に実行されます。

                現在いずれかのキーが押下されているかはisPressed変数に、押下されているキーはkeyに、固有番号はcodeに、押下されているキーで入力される文字はcharに格納されます。

サンプル
