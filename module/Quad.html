name

名前

Quad

Description

説明

                ウィンドウに四角形を描画するために使用されます。

                頂点となる任意の4点のX,Y座標x1,y1,x2,y2,x3,y3,x4,y4からなる四角形を描画します。Rectangleとは異なり、4点全てを任意に決めることができます。

                図形の色はfill関数で、塗りつぶしなしはnoFill関数で指定することができます。
                枠線の色はoutlineFill関数で、枠線なしはnoOutline関数で指定できます。
                枠線の太さはoutlineWidth関数で指定可能です。

                四角形の位置は座標を指定する以外にrotate関数を使用することで変化させることができます。ウィンドウの左上座標0,0もしくはchangeBasePoint関数で任意に設定された座標を中心とした円周上を移動します。また、移動時に角度も変化します。角度のみを変化させたい場合はchangeBasePoint関数で四角形の重心座標を指定してください。
