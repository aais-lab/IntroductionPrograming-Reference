name

名前

Text

Description

説明

                ウィンドウ上に任意のテキストを表示するために使用されます。

                指定されたテキストはOSのデフォルトフォント（MacではHelvetica、WindowsではSegoe UI）20ptで表示されます。
                フォントカラーはblackで表示されますが、fill関数で変更可能です。
                フォントを変更したい場合はfont関数で変更可能です。
                また、使用可能なフォントはavailableFonts関数で確認できます。

                指定したX座標、Y座標は表示されるテキストの中心座標になります。
                テキストはウィンドウを超えても1行で表示されますが、改行コード\nを含む文字列の場合は改行コードの位置で改行されます。
