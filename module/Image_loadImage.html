name

名前

loadImage

Description

説明

                ウィンドウに表示する画像を読み込むために使用されます。
                使用可能な画像フォーマットはPGM、PPM、GIF、PNG、XBMのいずれかのみです。
                それ以外の画像フォーマットを選択した場合や拡張子がファイルパスに含まれていない場合はFileTypeErrorになります。

                @animationデコレータでTrueを設定した場合、表示されている画像は再描画されるため、各画像フォーマットでアニメーション画像を表示したい場合は注意が必要です。

                loadImage関数な内部的にImageオブジェクトを生成し、描画の基準となる座標点anchorを画像中央centerに設定します。
                描画の基準座標点の変更はchangeAnchor関数で、画像の拡大縮小はscale関数で、画像の描画はshow関数で可能です。
