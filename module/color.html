category

Color

name

名前

color

Description

説明

                3値で色を指定するために使用されます。

                現在のCOLOR_MODEによって指定可能な値の範囲が異なります。COLOR_MODEがRGBの場合、整数値0〜255、実数値0.0〜1.0の範囲です。COLOR_MODEがHSVの場合、整数値0〜100、実数値0.0〜1.0の範囲です。いずれの場合も整数値と実数値を混在させて指定することはできません。
                負の数を指定した場合、範囲外の値を指定した場合など色に変換できない値が指定されたときはColorErrorが送出されます。
