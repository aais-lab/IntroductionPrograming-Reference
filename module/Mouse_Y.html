Introduction Programing Reference

category

Mouse

name

名前

Y

Description

説明

                ウィンドウ上にあるマウスカーソルの現在座標のうちY座標をint型で取得できます。

                初期座標は-100で、カーソルがウィンドウ上に載ったタイミングで更新されます。また、ウィンドウ外にマウスカーソルが移動した場合、最後にウィンドウ上にあった座標を保持します。
                座標はウィンドウの左上隅を基準点（X, Y = 0, 0）とする絶対座標系です。

Examples

サンプル

                        
                        
                            from IP import *

Syntax

構文

  • mouse.Y

Parameters

引数

  • None

Return

戻り値

int型の現在のマウスカーソルのY座標
