name

名前

@mouseReleased

Description

説明

                マウスの何らかのボタンを押して離した際に処理を行う関数に付与するデコレータです。
                このデコレータを付与した関数は、マウスの何らかのボタンを押して離した際に実行されます。
                押下したタイミングで実行する場合は@mousePressedを付与する必要があります。

                現在マウスボタンが押下されているかはisPressed変数に、押下されているボタンはmouseButtonに格納されます。
                押下されたウィンドウ上の座標はpressX、pressY、離されたウィンドウ上の座標はclickX、clickYに格納されます。
