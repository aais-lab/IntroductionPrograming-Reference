category

Window

name

名前

Window

Description

説明

                図形などを描画するウィンドウを表示するために使われます。
                show関数を実行すると、その時点までに生成された図形などを描画済みのウィンドウが表示されます。
                
                ウィンドウの大きさはsize関数で、幅width高さheightを指定できます。
                上限値は幅width1000px、高さheight700pxです。上限を超えて指定した場合、上限値が適用されます。

                背景色backgroundはカラーコード（例#000000）、色名（例red）またはcolor関数を使って指定できます。画像は指定できません。

                ウィンドウの大きさの変更はsize関数、タイトルの変更はtitle関数、背景色の変更はbackground関数でできます。

Examples

サンプル

                        
                            import IP

                            window = IP.Window()
                            window.show()
                        
                    
                        
                            import IP

                            window = IP.Window(500,500,"white")
                            window.show()

Syntax

構文

  • Window(width,height,backgroundcolor)

Parameters

引数

  • width

    (int)

    1〜1000

    ウィンドウの幅

  • height

    (int)

    1〜700

    ウィンドウの高さ

  • background

    (color)

    ウィンドウの背景色

Return

戻り値

void
