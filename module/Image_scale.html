name

名前

scale

Description

説明

                読み込んだ画像の倍率を変更するために使用されます。

                拡大する場合と縮小する場合は指定する値の範囲が異なります。
                縮小する場合は0より大きく、1より小さい値を浮動小数点数floatで指定してください。縮小率と解釈され、指定されたratioの割合で画像のアスペクト比を保ったまま縮小されます。
                拡大する場合は1より大きな値を整数intで指定してください。拡大倍率と解釈され、指定されたratio倍に画像のアスペクト比を保ったまま拡大されます。
                0及び1の場合は、縮小率0、拡大倍率1と解釈され、画像の大きさに変化はありません。

                倍率によって画像サイズを変更するため、画像サイズの微調整を目的としたscale関数の使用は推奨されません。
