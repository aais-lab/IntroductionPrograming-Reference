Keyboard

name

名前

keyboard.code

Description

説明

                押下されたキーの固有番号が格納されています。
                初期値は空白文字列です。

                この変数は、キーが押下されたタイミングで更新されます。
                そのため、一度押下して離した場合でも次のキーが押下されるまでは値を保持し続けます。

                キーの固有番号は規則性のない数字であるため、使用は非推奨です。
