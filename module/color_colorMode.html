category

Color

name

名前

colorMode

Description

説明

                使用する式空間COLOR_MODEを変更するために使用されます。

                対応している式空間はRGB,HSVのみです。初めはRGBが設定されています。
                RGB,HSV以外を指定した場合、ColorErrorが送出されます。
